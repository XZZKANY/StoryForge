# 新建卷 / 新建章节（已验证真实路径）

通用：query 带 aid=2503&app_name=muye_novel；写 body 为 form-urlencoded；写操作需 header x-secsdk-csrf-token（webview 内 fetch 自动带，见 写侧架构.md）；a_bogus/msToken 不需要。

## 建卷（可选，书默认已有「第一卷：默认」）
- POST /api/author/volume/add_volume/v0?aid=2503&app_name=muye_novel
- Header: x-secsdk-csrf-token
- Body: book_id=<id>&volume_name=<名>&volume_desc=<可空>

## 列卷（拿 volume_id）
- GET /api/author/volume/volume_list/v1?aid=2503&app_name=muye_novel&book_id=<id>
- 返回 data.volume_list[] = {index, book_id, volume_id, volume_name, item_count, can_delete}

## 新建章节（建空草稿）
- POST /api/author/article/new_article/v0/?aid=2503&app_name=muye_novel
- Body: aid=2503&app_name=muye_novel&book_id=<id>&need_reuse=1
- 返回 code:0；item_id 见编辑器 URL /publish/<item_id>，或 chapter/draft_list 查询
- 实测不带 a_bogus/msToken 返回 code:0。

## 存正文（已实测）
- POST /api/author/article/cover_article/v0/?aid=2503&app_name=muye_novel
- Body: aid, app_name, book_id, item_id, title, content(HTML <p>), volume_name, volume_id
- 返回 {code:0, data:{latest_version}}
- （edit_article 是 GET 加载草稿；发布见 章节发布.md）
