## 🎯 使用指南

### 启动应用
```bash
cd D:\StoryForge\apps\desktop
pnpm tauri dev
```

### 打开测试项目
1. 点击"打开项目"按钮
2. 选择 `D:\test-storyforge-project`
3. 文件树显示 4 个 Markdown 文件

### 编辑文件
1. 点击文件树中的文件
2. 在编辑器中修改内容
3. 按 Ctrl+S 保存
4. 状态栏显示"已保存"

### 新建文件
1. 点击"新建文件"按钮
2. 输入文件名（如 chapter-004.md）
3. 自动打开新文件编辑

### 使用菜单
- **File** → 文件操作
- **Edit** → 编辑操作
- **View** → 视图控制
- **Help** → 帮助文档
