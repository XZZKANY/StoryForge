# Phase 9B 真实 LLM 冒烟 3 章

## 第 1 章 真实冒烟 1

雾港的雨带着海腥气，盐粒打在林岚的冲锋衣上，发出细密的沙沙声。他右臂夹紧工具包，左手虚握成拳，垂在身侧。雨是从傍晚开始下的，不大，但足够把灯塔外墙的石头浸得湿亮。

锈蚀的金属梯紧贴塔身，向上延伸进灰白色的雾里。林岚开始爬。梯级间距比他记忆中窄了半寸，冰凉的触感透过手套渗进掌心。每爬三级，他左肩便有一个细微的滞涩感，旧伤在潮湿空气里苏醒。他停了一下，换右臂使力，继续向上。风从塔身与梯子的缝隙钻进来，带着底下礁石拍岸的闷响。

塔顶机房的门没锁。林岚推开时，门轴发出一声短促的呻吟。里面比外面暖，混着机油、海雾和老旧电气的微焦气味。控制台在靠窗的位置，绿色指示灯在昏暗里规律地明灭。一切正常，除了灯头旋转时发出的声音——那该是均匀的、低沉的“嗡——嗡——”声，此刻却夹杂着极轻微的“嗒”，间隔不规律，像心律不齐。

林岚放下工具包，从侧面面板拧下四颗螺丝。他动作很慢，每转一圈，耳朵都贴近金属面板半寸。螺丝落在地上，声音被地毯吸收。他拉开面板，露出里面纠缠的线路。肉眼可见的地方都覆盖着薄薄的盐垢。他戴上头灯，光柱聚焦在谐振器和继电器上。一切如常。

他拿出掌上审计终端，一个老旧型号的黑色设备，屏幕有划痕。开机需要十秒。他在这十秒里，伸手触碰了一下主频振荡器的外壳。很烫。这不对，塔内有通风系统，温度不该积聚在这里。他缩回手，指尖皮肤上留下一片干燥的灼热感。

屏幕亮起。他调出信号频谱监测程序，将微型探头夹在谐振器输出端。数据流开始滚动。正常的频谱峰值应该平稳，像一座小山。现在，那座山的轮廓在颤抖，每隔十几秒，就会有一个不该存在的尖刺脉冲，短暂地冲上又跌落。脉冲很微弱，混在背景噪声里，但审计终端把它标成了红色。

林岚把探头移到不同的监测点。供电单元正常。温度传感器正常。发射功率在允许范围内。只有频率稳定性指数，在缓慢下滑，像沙漏里流走的沙。

他收起终端，把探头和数据线一根根理好，放回工具包。然后他重新拧上面板螺丝。拧螺丝时，他的呼吸和扳手的转动节奏一致，很轻，很长。锁好面板，他走到窗边，推开一扇小窗。风立刻灌进来，雨丝斜打在他脸上，冰凉。他向下看，港口的灯光在雨雾里晕成一片模糊的黄。码头边，那艘“顺昌”号货轮的轮廓勉强可辨，黑色的船体上没有光。

他关上窗，在控制台前坐下。调出审计链记录界面，界面是黑底绿字，很简陋。他在第一行输入：“时间：2023年10月17日，21:43。地点：雾港一号灯塔，塔顶机房。现象：发射信号频率稳定性异常，存在非周期性微弱杂散脉冲。环境：塔内谐振器外壳过热，体感约45摄氏度以上。外部持续降雨，相对湿度极高。”输入完毕，他按了确认键。数据被加密，发送到某个遥远的服务器。

楼下传来脚步声，沉重，沿着旋转铁阶向上。林岚没有动，只是把审计终端滑进外套内袋。机房门被推开。进来的是灯塔维护员，一个五十多岁的男人，姓周，身上带着浓重的烟草和廉价肥皂混合的气味。他头发湿了，贴在额头上。

“林调查员，”周维护员喘着气，手撑在膝盖上，“您还在啊？信号有什么问题吗？我们每天检查，设备都挺正常的。”他说话时，眼睛快速扫过控制台和林岚放在一旁的工具包。

“例行检查。”林岚说，声音平稳，“你听到过什么特别的声音吗？”

“声音？”周维护员直起身，擦了把脸，“这铁家伙一转起来，什么声音都盖住了。耳朵早不行了。”他嘿嘿笑了笑，露出烟渍牙。

“上次更换谐振管是什么时候？”

“记录上是去年九月。不过……”周维护员顿了顿，摸出烟盒，抖出一根点上，“不过那根管子是旧的，从库房底翻出来的，当时新的还没到货。用了这一年多，我看也没啥事。”他吐出一口烟，烟雾在潮湿空气里散得慢。

林岚看着他。“库房现在还有这种旧管子吗？”

“早没了。就那一根。”周维护员吸了口烟，烟头亮了一下，“林调查员，您是担心信号不好，船找不着路？这大雾天，船都停着呢。等雾散了，我再多调调发射角度，肯定行。”

“你调过角度？”

“小调。就拧了拧底座固定螺栓，紧一紧。风大雨大的，怕它松了。”周维护员弹了弹烟灰，灰烬落在地上，“这雨一下，盐分重，什么金属都得上锈。我跟上面报过，申请点防锈油，到现在也没批。”

林岚没再问。他拿起工具包，走向门口。“今天先到这里。”

“这就走？”周让开路，靠在门框上，“要不要喝口水？我那儿有热茶。”

“不用。”林岚侧身出去，左臂小心地避开了门框。

他顺着铁阶往下走，脚步声被旋转的结构吞没。走到一半，他停下，侧耳。只有风声和雨声。他继续走，出了塔基的门。雨下得更大了些，打在冲锋衣兜帽上，噼啪作响。他走到码头边一处避雨的栈桥下，从内袋摸出审计终端，屏幕湿漉漉的。他用衣袖擦干，调出刚才的记录，在下面新增了一行。

“询问：维护员周。供述：去年九月更换谐振管，但使用旧库存部件（型号不明）。近期曾调整发射塔底座螺栓（非标准作业）。提及塔内设备缺乏常规防锈维护。备注：其知晓信号异常可能，但归因于天气与设备老化。”他顿了顿，又加了一句：“其陈述时呼吸频率稳定，无显著应激反应，但主动提供过多细节（调整角度、报修防锈油）。”

保存。发送。加密标识闪烁了一下，确认传输完成。

林岚关掉屏幕。栈桥下的风带着水汽，冰凉地钻进他没有拉紧的衣领。他站在原地，看着远处海面上“顺昌”号沉默的黑影。过了一会儿，他抬起右手，食指和拇指轻轻按了按左侧锁骨下方的位置，那里的旧伤在阴雨天总会隐隐发酸。然后他放下手，拉紧冲锋衣拉链，朝着码头上唯一还亮着灯的值班室走去。

## 第 2 章 真实冒烟 2

值班室的窗户蒙着水汽，光晕昏黄，从里面漏出来。林岚推门时，门锁有些涩，他加了点力，锁舌咔嗒弹开。室内比外面暖，一股潮湿的烟草味混合着旧木头和油墨的气息扑面而来。只有一盏老式煤油灯摆在桌上，玻璃罩被熏得发黑，火焰稳定地烧着，发出极其轻微的嘶嘶声。

“林调查员？”坐在桌后的人抬起头。是个五十岁上下的男人，穿着浆洗发白的灯塔工作服，袖口磨出了毛边。他手边放着一个搪瓷缸，热气氤氲。

“老郑。”林岚点头，算是打了招呼。他把湿透的冲锋衣领口拉开一点，让冰冷的空气接触颈部的皮肤。

“外头雨大。喝口热水？”老郑用下巴点了点旁边的暖水瓶，“刚烧开的。”

“不用。”林岚走到窗边，用袖口擦掉一片水汽。视线穿过玻璃，码头上一片漆黑，只有远处引航船的灯塔在雾中划出一道微弱的光柱。“顺昌”号那庞大的黑色轮廓几乎要与夜色融为一体，只有船舷上几处反光标记，证明它的存在。

“那船停了三天了。”老郑的声音从背后传来，带着一种见怪不怪的平直语调，“说是等天气，我看不像。船头对着外海，可主机没动静。昨天半夜我听见动静，以为要走，结果只是挪了个泊位，又停下了。”

林岚没转身。“船员上岸了？”

“零星几个。”老郑喝了口水，水声很响，“昨天下午有三个，去了码头东头的‘老地方’大排档。点了些海货，喝了酒，说话声音很大。用的不是本地方言。”

“你还记得他们聊什么？”

“隔着雨，听不全。”老郑用手指沾了点茶水，在油腻的桌面上画着什么，“好像提了一嘴‘验货’、‘潮水’，还有……‘信号’。”他最后两个字说得很慢，像是在回忆，“当时没在意。现在您问，才觉得有点怪。”

林岚转过身。“具体是哪两个字？‘信号’？”

“就这个词。原话好像是……‘那边的信号还准不准’。”老郑抬头看他，“我耳朵也不好使了，雨声大。”

林岚走到桌前。桌上除了煤油灯和搪瓷缸，还摊着一本厚厚的航行日志，纸页泛黄卷边。日志旁边放着半包拆开的烟，和一盒火柴。“他们三个，有什么特征？”

“一个光头，脖子上有纹身，像条船锚。一个瘦高个，戴眼镜。还有一个……”老郑眯起眼，“背有点驼，手指很长，指甲缝里黑乎乎的，像是干机械活的。他们走的时候，光头那个回头看了看灯塔方向，就一眼。”

“看灯塔？”

“嗯。然后催着另外两个走了。往东边，渔具店那个方向。”老郑用手指了指东面，“那条街晚上没灯，黑得很。”

林岚的目光落在航行日志上。“昨天的记录，我能看一眼？”

老郑把日志推过来。“您随便看。反正就是些风啊浪啊，船进船出，没什么特别。”他站起身，走到角落的铁皮柜前，拿出一个暖水瓶，开始给自己续水。

林岚翻到昨日那一页。字迹工整但略显潦草，记录着风力、风向、能见度、进出港船只。他的手指停在“顺昌”号那一行。记录是：“23:00，‘顺昌’号货轮（巴拿马籍），移泊至七号锚位，未申请离港。船员三人上岸，未归。备注：船方未提供货物更新清单。”

他拿出审计终端，开机。屏幕亮起，他调出审计链界面，在“物证关联”栏输入：“值班室航行日志记录，‘顺昌’号行为异常：移泊未报、人员上岸未归、缺失货物清单。关联上文‘频率稳定性异常’及维护员周提及‘旧谐振管’。”

保存。加密。发送。整个过程不到一分钟。

老郑端着水杯走回来，看了一眼他手中的黑色设备。“您这东西，挺利索。”

“记录工作。”林岚把终端收起。“昨天那三个上岸的船员，之后没再出现？”

“没有。船也没开。今早上我还特意去看了眼船员名单，贴在船长室门口。名单上写着十二个人。可我守这码头，最近一次看见他们批量上岸，是一个月前。”老郑吹着杯口的热气，“有些船，规矩乱。人货分离，各走各的，也是有的。”

林岚走到门边，拉开门。风雨的气息重新涌进来，带着咸味和凉意。“谢谢。”

“这就走？”老郑有些意外，“这雨一时半会儿停不了，外头路滑。”

“没事。”林岚侧身出去，左手在关门时虚扶了一下门框，避免左臂完全伸直。门在身后合上，煤油灯的光晕被彻底隔绝。

他站在值班室外，雨点立刻密集地打在他的肩上和兜帽上。远处“顺昌”号的方向，浓雾像一堵灰色的墙，把船吞没得更深。他拉上拉链，沿着码头边缘的防波堤向东走去。脚下是湿滑的混凝土，海水拍打堤岸的声音沉闷而有规律，一下，又一下。

走了约莫五十米，前面出现一条狭窄的巷口，没有路灯。巷口第一家招牌褪色严重，依稀能辨认出“渔具”两个字。再往里，一片漆黑。林岚停下脚步，站在巷口外的阴影里。雨水顺着冲锋衣的衣摆滴落，在脚边汇成细小的水流。

他从内袋里再次摸出审计终端，屏幕在雨幕中泛着微光。他调出地图界面，找到“老地方”大排档的位置——就在巷子深处，靠近海边。然后他新建了一条记录：

“时间：2023年10月17日，22:15。地点：雾港码头东侧渔具店巷口。关联人物：‘顺昌’号船员三人（特征：光头、纹身、瘦高眼镜、驼背机械工）。行动轨迹：疑向东侧‘老地方’大排档方向移动，进入无照明区域。当前环境：雨势未减，能见度低于二十米。下一步：确认目标人物最终落脚点及接触人员。”

保存。发送。屏幕上的绿色文字闪烁了一下，确认记录已上传至审计链。

他关掉屏幕，将终端塞回内袋最深处，确保拉链拉严。然后他抬手，将兜帽的抽绳拉紧，只露出一双眼睛。他最后看了一眼漆黑的巷子深处，那里只有雨声和风声，以及更远处隐约传来的、被海雾闷住的海浪撞击礁石的闷响。

他迈步走进了巷子。

## 第 3 章 真实冒烟 3

巷子里没有光。雨声把其他声音都压得很低，只剩下水流沿着两侧墙壁淌下来、汇入脚下积水的汩汩声。空气里有鱼腥味，混着旧木板和受潮石灰的霉味。林岚的靴子踩在坑洼不平的石板上，水花溅到小腿肚，冰凉。

巷子尽头透出一点昏黄的光晕，被雨幕和浓雾切割得模糊。光晕来自一扇半开的木门，门框上方挂着一块歪斜的木牌，上面的字被海盐腐蚀得难以辨认。门里传出压低了的人声，听不清词句，语调带着抱怨。

林岚没有直接走向那扇门。他侧身贴在湿冷的墙上，慢慢向门边移动。左臂的旧伤在冷雨里一阵阵发紧，他用右手扶着墙壁，保持平衡。门缝约莫一尺宽，里面是一条狭窄的过道，尽头似乎连着另一个空间。过道墙壁上挂着些渔网和塑料浮标，水珠顺着网线往下滴。

“……信号还他妈准不准？潮水都要变了。”一个沙哑的男声从里面传来，带着明显的不耐烦。

“急什么，老刘不是说再等等？”另一个声音稍显年轻，但语速很快，“雾不散，出去就是瞎子。”

“等？再等下去，船都快他妈烂在这鬼港口了！”先前说话的男人声音拔高，又迅速压下去，“那玩意儿到底搞不搞得定？上次调那破天线，差点把老子的手拧断。”

“老刘说他有办法……”

“他有办法？他有个屁办法！灯塔那老东西一天到晚嗡嗡响，我耳朵都快聋了，信号还是时有时无。要我说，干脆……”

“嘘！”第三个声音响起，很低，带着不容置疑的阻断意味，“隔墙有耳。去里屋说。”

林岚的呼吸很轻。他看着过道尽头，那里似乎有一道布帘，声音就是从帘子后面传来的。他左手虚握成拳，垂在身侧，右手摸向内袋里的审计终端，但没有拿出来。脚步声响起，三个人，踩着木地板，走向更深处。接着是布帘被撩开又落下的窸窣声，声音立刻变得模糊沉闷。

他等了十几秒，确认没有更多动静，才闪身进了那扇门。过道里气味更重，除了咸腥，还有浓烈的酒精和廉价烟草燃烧后的焦苦味。地上散落着几个扁平的啤酒罐，被踩得变了形。他走到布帘前，帘子是厚重的帆布，边缘磨损。他抬手，用两根手指轻轻掀开一道缝隙。

里面是一个小隔间，面积不大，摆着一张油腻的木桌和几把椅子。桌上亮着一盏光秃秃的灯泡，没有灯罩，光线刺眼，把三个人的脸照得苍白。光头、纹身的那个背对着门口坐着，肩膀很宽。瘦高眼镜的男人面对门口，脸色在光线下显得有些灰败。驼背的那个侧对着，正低头用一把螺丝刀撬着桌面上的一个黑色方盒，动作熟练，指甲缝里确实是深色的污垢。

桌上摊着一张海图，边缘卷曲。旁边放着几个湿漉漉的塑料袋，袋口敞开，里面是深色的、形状不规则的块状物。还有一个打开的笔记本，纸页被水汽浸得发皱。

“老刘明天还不来？”瘦高个推了推眼镜，镜片反光。

“说信号没调好，他出不了门。”驼背的男人头也不抬，继续手里的动作，“灯塔那边看的紧，白天有巡查的。”

“巡查个屁，就一个糟老头子。”光头嘟囔着，拿起桌上一个金属水壶灌了一口，“那东西赶紧弄好。这批‘货’不能再放了，再放就真成臭鱼了。”

“催什么催。接口不匹配，得改。”驼背男人把螺丝刀插进缝隙，用力一撬，咔嗒一声轻响，方盒盖子弹开。里面是密集的线路板，还有一块屏幕。屏幕没有亮。

林岚看着那块屏幕。他的右手在帆布帘外缓缓收紧，指节绷紧。

光头探过头去：“能修好？”

“试试。总比天天爬那个锈梯子强。”驼背男人从裤兜里掏出一个更小的、巴掌大的黑色设备，开始连接线路。他的手指移动得很快，带着一种惯常的熟练。

林岚放下帘子，后退半步。他退出过道，回到外面的门边。雨水立刻打在他脸上。他转身，看向巷口方向，又看了看这扇门旁边的墙壁。墙上靠着一架锈蚀的铁梯，通向二楼一个黑黢黢的窗口。梯子很窄，锈迹斑斑。

他走到梯子前，没有立刻上去。他从内袋摸出审计终端，开机。屏幕亮起，他调出审计链，在“现场勘查”栏输入：

“时间：2023年10月17日，22:35。地点：雾港码头东侧，‘老地方’大排档后厨入口内过道。发现‘顺昌’号船员三人（光头、瘦高眼镜、驼背机械工），于内间操作疑似信号干扰或劫持装置（黑色方盒，内有电路板及未知接口）。三人对话提及‘信号不准’、‘潮水’、‘货’、‘灯塔锈梯’。初步关联：信号异常可能与人为干扰有关，干扰源疑为船上人员，且存在未申报货物（‘货’，疑指渔获或走私品）。”

保存。发送。加密标识闪烁。

他关掉终端，收好。然后他抬手，抓住铁梯最低一级横档，用力试了试锈蚀程度。金属发出细微的呻吟。他左脚踩上去，开始向上爬。雨打在梯子上，声音沉闷。爬到二楼窗口高度，他停下来，侧耳。里面有模糊的谈话声，还是刚才那三个人。他凑近满是污垢的窗口玻璃，用袖口擦开一小块。

窗口对着一个堆满杂物的仓库。几个人影在晃动。光头男人正在用脚踢着一个麻袋，麻袋裂开，露出里面深色的、湿漉漉的东西，形状不规则，散发着浓重的腥味。不是普通的渔获。气味里混杂着一种陌生的、微甜的腐败气息。

驼背男人在另一个角落，摆弄着一台小型的柴油发电机，发电机突突地响着，声音被雨声盖住大半。瘦高个蹲在地上，往一个硬塑箱子里装东西，动作很快。

林岚看着硬塑箱子侧面喷的字迹，模糊，但能辨认出“冷藏-3”和“生物制品”的字样。这不是常规渔船会用的箱子。

他记下箱子编号的前几位。然后他的目光扫过仓库地面，在发电机的光线下，他看到地面有一片颜色更深的水渍，从一个角落蔓延出来，形状不规则。水渍边缘，有极细微的、反光的东西，像盐粒，又像别的什么结晶。

他离开窗口，顺着梯子原路返回。下到地面时，他的左脚踩滑了一下，身体晃了晃，右手迅速抓住梯档稳住。雨更大了，砸得他肩头生疼。

他退回到巷口阴影里，再次拿出审计终端。

“补充记录：二楼仓库内，发现疑似非法加工或冷藏的生物制品（气味异常，包装箱标识为‘冷藏-3’、‘生物制品’）。存在小型发电机。地面有可疑水渍及反光结晶物。关联：一、异常‘货’可能为高度敏感或禁止流通品；二、使用发电机可能为维持特定冷藏条件，或为信号干扰设备供电（需核实）；三、气味及结晶物待进一步鉴定。”

保存。发送。

他盯着屏幕上闪烁的绿色光标，然后输入最后一行：“目标已明确。干扰源在‘顺昌’号仓库，由三名船员操作。建议：固定证据，必要时介入。风险：目标警觉，可能转移或销毁证据。环境：大雨，高湿度，证据易损。”

确认。传输。加密标识稳定亮起。

林岚关掉屏幕。他站在雨幕中，看着巷子深处那扇透出昏黄光晕的门，又抬头望向远处浓雾笼罩的海面。“顺昌”号的黑影在雨雾中时隐时现。他右手按了按内袋，确认审计终端的位置。然后他拉紧兜帽，转身，朝着码头值班室的方向走去。脚步很稳，踩在积水里，发出规律的啪嗒声。
