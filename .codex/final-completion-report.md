## 🔧 修复的问题

### Windows uvloop 兼容性
**问题**：`uvloop` 不支持 Windows
**解决**：改用 `.venv/Scripts/python.exe` 直接启动 uvicorn

```rust
// 修改前
Command::new("uv").args(&["run", "uvicorn", ...])

// 修改后
Command::new(".venv/Scripts/python.exe")
    .args(&["-m", "uvicorn", ...])
```
