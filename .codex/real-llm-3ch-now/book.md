# Phase 9B 真实 LLM 冒烟 3 章

## 第 1 章 真实冒烟 1

雾港的早晨像一块泡软的灰布，海雾贴着码头往里爬，木桩上挂着咸水，绳结一碰就发出干涩的摩擦声。灯塔在远处立着，塔顶的光每隔几秒扫过港湾，却没有照出该有的节奏，半截光束在雾里断开，像被人用刀切了一口。

林岚站在防浪堤边，风把她外套下摆扯得贴住腿。她把左臂收在身侧，袖口压得很平，只用右手托着便携接收器。屏幕上，频谱线来回抖动，几个尖峰挤在一起，像被硬塞进同一格。她盯了两秒，把时间戳点进审计链，输入：灯塔信标，07:14，首轮异常。

接收器里传出一阵轻响，像砂纸磨过金属。她抬头看灯塔，塔身外壁渗着潮气，白漆底下露出一层旧灰。海风带来柴油味，夹着一丝焦黑的线皮味，细得几乎压在舌根上。

“今天第几次跳号？”她问。

码头管理员老邱搓着手走近，靴底踩过积水，啪地响了一声。“第三回了。昨晚更糟，灯扫到一半卡住，像有人掐了脖子。”

林岚没有接话，只把接收器递过去。老邱没敢碰，只凑近看了眼屏幕，喉结动了动。“这波形不像海雾干扰。”

“不是。”林岚说。

她蹲下身，指尖擦过堤沿的湿漆，摸到一粒细黑的粉末。她把样品装进取证袋，封口后扫了一遍条码，记录入链：堤边残留，黑色颗粒，疑似绝缘材料。袋子贴上皮肤时带着一点潮冷，拉链的金属扣硌在指腹上，发出轻微的咔哒声。

老邱往灯塔方向看了一眼，声音压低了些。“昨夜值班的人说，塔里有烟。”

“谁说的？”

“值守的魏修。他说闻见了，先是焦味，后头连门缝都透灰。”

林岚抬手看表，秒针走动的声音细而脆。她把这句话写进审计链，附上证词来源、时间、地点，签名确认后，屏幕右下角跳出一道绿色校验线。

“人呢？”

“在塔下维修间。”老邱顿了顿，“锁着门，不肯开。”

林岚收起接收器，沿着堤坝往灯塔走。脚下石阶覆着薄水，鞋底一压，水便从石缝里挤出来。越靠近塔身，那股焦味越重，像线圈烧到发白后的味道。门外挂着一盏小灯，灯罩上蒙着盐霜，光线黄得发浑。

维修间的门板里传出两下钝响，像有人在里面搬箱子。林岚停在门口，没有敲门，先把门牌号、封条状态、门缝烟痕逐项拍照，照片一张张上传，自动落入审计链。她举起手电照向门缝，灰白的烟丝从底边慢慢溢出来，细得像被揉碎的纸灰，碰到冷空气便散开。

“开门。”她说。

里面安静了片刻，随后是椅脚拖地的刺耳声。门链先响，接着锁芯转动。门开了一道窄缝，热气裹着焦味冲出来，扑在她脸侧。魏修站在里面，额头全是汗，工装领口湿了一圈，手里攥着一截烧黑的电缆。

“别进。”他说，“塔上那台机子，不对劲。”

林岚把手电移过去。维修间靠墙摆着拆开的发射箱，铜线盘卷得乱，机壳内侧有一层新鲜烟渍，边缘还挂着未熄尽的黑屑。她伸出右手，用镊子夹起一小段熔化的绝缘皮，放进样本盒。盒盖合上时，发出一声清脆的扣响。

“什么时候烧的？”

“昨晚十一点后。”魏修抹了把脸，掌心蹭过下巴上的灰，“我按规程停机，灯还是在扫。信号没断，反而多出一段脉冲，像有人在塔里回播。”

林岚看着发射箱上那枚被烫出白点的编号牌，伸手把铭牌角度拍正。照片上传后，她低头核对波形和现场记录，把“异常信号与设备热损伤同步”写进审计链，提交的那一刻，屏幕上跳出确认码，反光映在她瞳孔里，冷得很稳。

外头的雾压得更低了，海面传来轮船汽笛，声音被雾切成两段。林岚把接收器重新贴近耳侧，里面有一串短促的电流噪音，像有人在远处连续敲击金属。她抬眼看向灯塔，塔顶那束光又扫过来，这次在半空停了半拍，随后才挪向海湾。

“塔上还有人。”她说。

魏修往后退了一步，后背撞上工具架，扳手碰撞出一阵哐啷。“我没看到。”

林岚把封好的样本袋收进内袋，拉链拉到底，手指在拉头上停了停。她转身往塔门走，靴底踩过门前积水，水面立刻起了一层细密的波纹。门内的楼梯向上延伸，潮气、铁锈和焦味拧在一起，顺着楼梯间往上爬。

她按下终端，最后补上一条记录：现场已确认，信标失真，存在烧灼痕迹，疑似人为干预。发送后，审计链亮了一瞬，像一根细线在雾里绷紧。

林岚收起终端，推门进了塔内。

## 第 2 章 真实冒烟 2

我先把灯塔内部的追查写实化，往上层设备间推进，确保把异常信号的源头、可入链的证据和关键人物都落到场面里。塔内比外头更潮。木阶吸着海水，踩上去发软，铁扶手冰得像刚从井里捞出来。林岚贴着墙上楼，靴底擦过灰，带出一串细响。每隔几级，就有一股焦线味从上面压下来，混着旧油和潮霉，呛得人喉咙发紧。

二层平台挂着一盏红灯，灯罩里积着盐。控制箱敞着，螺丝刀横在边上，刀柄上粘着一层黑灰。箱门内侧贴着值班表，纸边卷起，日期被人用铅笔重描过三次。林岚举起手电照进箱里，主线后面多出一组副接点，细铜线从底座绕出去，钻进墙缝。电阻盒外壳烫得发白，缝口还吐着一丝烟，细得像线头。

她蹲下，取证袋贴着膝盖发凉。镊子夹起一片熔化的绝缘皮，装袋，封口，扫码。屏幕亮了一下，记录栏跳出新条目：塔内副线路，编号缺失，外壳灼痕新鲜。她把手电移开，照见墙角一只铁桶，桶底堆着灰烬，灰里压着半截火柴和一枚烧黑的铜钮。

楼上传来一声闷咳。

林岚抬头，灯泡轻轻晃了一下。她没有出声，沿着平台往上走。顶层机房门虚掩，门缝里透出一线白光，里面有金属碰撞的轻响，短而密，像有人用指甲敲着扳手。

她一把推开门。

屋里站着一个男人，背对着门，手还按在发射台上。台面上摊着两张潮湿的调拨单，墨迹被汗水洇开，边角盖着港务局红章。男人听见动静，手指一缩，扳手掉在地上，砸出清脆一声。

“别动。”林岚说。

男人转过身，脸上沾着煤灰，袖口卷到手肘，指节有新烫出的红印。他盯着她手里的终端，喉结滚了一下。“我没偷改信号。”

“谁改的。”林岚把镜头对准调拨单，拍照，上传。

“单子上写得清楚。”男人朝桌上一点，“灯塔频率校准，临时加装烟扰器。签字的人不是我。”

林岚低头扫过签名栏，字迹被水泡开一截，末尾还能看出一个“梁”字。她把两张单据分开，按页拍下编号、印章、折痕，逐项写进审计链。屏幕右下角的校验线一条条亮起，机房里只剩风扇转动的低鸣。

“谁送来的。”她问。

男人沉了半秒，从工具箱底抽出一枚黄铜钥匙牌，递到她面前。牌面磨得发亮，背面刻着“北栈七码头”。“昨晚十点后，一个穿雨衣的女人送来的。她没上楼，只把这个丢进值班室。她说，照着做，灯就不会停。”

林岚接过钥匙牌，指腹擦过刻痕，边缘割得发涩。她把它装进证物袋，封条压平，扫码入链。袋子贴上掌心，带着一点残余热度。

“她什么样。”她说。

“个子高，左手戴黑手套。”男人看着她的左臂，声音低了一点，“她说，灯塔要是响错三拍，港里那条线就能接上。”

楼下忽然传来一阵急促的敲门声，接着是魏修的喊声，被风扯得断断续续。机房角落里，旧风箱咔地响了一下，最后一缕烟从电阻盒缝里冒出来，贴着天花板散开。林岚把终端收进胸前口袋，转身扫过整间机房，铭牌、单据、焦痕、钥匙牌，一项不落地确认入链。

她抬手按下发送。屏幕亮白一瞬，审计链弹出回执。

门外的海雾撞在窗玻璃上，发出细碎的沙响。灯塔顶端的警铃又响了一声，短促，干哑。

## 第 3 章 真实冒烟 3

林岚把终端扣进胸前口袋，转身推门。

门外的风像一把湿刀，贴着脸刮过去。塔顶回廊窄得只能侧身，铁栏杆被海雾浸得发凉，掌心一按，冷意立刻钻进骨缝。她左臂在门框上蹭了一下，旧伤在肘后牵出一记钝痛，她没停，右手扶着栏杆往外走。

警铃还在作响，声音短，干，像喉咙里卡着砂。

回廊尽头立着一只黑箱，外壳钉着港务局封签，封条被热气烤卷了一角。箱侧接着一束细线，沿着塔身外壁钻进避雷针底座，线皮已经熏成褐色。林岚蹲下，手电照进缝里，里头不是常规发射模块，而是一套改过的脉冲回播器，旁边压着一截湿透的纸带，带面上印着灯塔信号的节拍。

她把镜头贴近，连拍三张，上传，标注时间、位置、设备编号。

纸带边缘沾着烟灰。她用镊子挑起一角，闻到一股焦胶味，底下还压着一点柴油渍。样本袋封口时，塑料片刮过指腹，发出细细一声响。她刚把袋子塞进内袋，身后就传来鞋底踩水的声响。

“别动那箱子。”

声音很平，带着雨气。

林岚没有回头，先把终端举起，镜头对准来人。雨衣的帽檐压得低，水珠从边缘往下滴，砸在铁板上，啪，啪，啪。那只黑手套还在，指尖捏着一枚港务局的工作牌，牌角磨白了。

“梁主任。”林岚说。

女人的手顿了顿，随后把工作牌收回袖口。“你认得我。”

“签名认得。”林岚把屏幕转过去，调拨单上的“梁”字被放大了，边角还留着昨夜水痕，“烟扰器也是你批的。”

女人盯着屏幕，没有伸手。“那张单子只管遮灯。箱子里那套东西，不归港务局。”

“归谁。”

“归码头外那条线。”她往回廊外侧抬了抬下巴，雾里有一盏远灯在晃，光色发白，“信号错三拍，北栈七码头就能接上。货船不会停，巡检也不会靠近。”

林岚把黑箱侧面的焊痕拍下，镜头一格一格扫过接线口。焊点新，锡面还发亮。她按下上传，审计链回执在屏幕上跳了一下，绿光把她指节照得发冷。

“谁让你做的。”她问。

女人没答，只抬起另一只手。那只手里攥着一截烧短的烟卷，烟灰被风吹得直落。她把烟头按灭在栏杆下沿，火星擦出一小点红，立刻被潮气压住。

楼下传来急促的脚步声，木梯被踩得咚咚响。魏修的声音从下面撞上来：“林调查员，码头巡检到了！”

紧跟着，另一道男声在楼梯井里回荡，带着哨音：“塔顶的人，站住。”

女人退了半步，雨衣下摆扫过地面，带起一串水线。她伸手去扯黑箱后的插头，林岚一步跨过去，右手按住她手腕，指骨顶在湿布上，凉得像铁。两人撞在一起，栏杆发出一声闷响。女人抬膝要顶，林岚侧身避开，左臂擦过栏角，旧伤一阵发麻，她顺势把人压向箱体，终端从胸前滑出半截，镜头仍亮着。

“别碰。”林岚说。

女人喘了口气，鼻息里带着潮烟味。“你拍完也没用。”

“已经进链了。”

她说完，抬手按下发送键。屏幕上跳出最终确认，黑箱编号、焊点、单据、钥匙牌、烟灰样本，一项项锁进审计链。回执弹出的那一刻，楼梯口的脚步声到了门边，铁门被人一把撞开，冷风卷着海雾灌进来。

魏修站在门口，脸色发白，身后跟着两名巡检。灯塔顶端那束白光正好扫过回廊，照在黑箱开裂的缝口上。缝里卡着一段铜线，末端烧得发黑，仍在轻轻冒烟。

女人盯着那股烟，喉咙动了一下，没再去摸箱子。她手腕还压在林岚掌下，雨水顺着袖口流到地面，滴成一小片深色。巡检上前，手铐碰在腰带上，叮的一声。

林岚松开手，弯腰捡起掉在地上的工作牌。牌面背后刻着一行细字，北栈七码头。她把牌装进证物袋，封口，扫码，发送。

海雾撞在玻璃上，发出细碎的沙响。灯塔的警铃停了，塔身只剩回风穿过铁架的低鸣。林岚收起终端，站在湿冷的回廊里，看着巡检把黑箱拆下，铜线一截截剪断，热焦味沿着风往海面散去。
