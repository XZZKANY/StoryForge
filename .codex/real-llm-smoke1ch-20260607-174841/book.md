# Phase 9B 真实 LLM 冒烟 1 章

## 第 1 章 真实冒烟 1

雾从海面爬上来，吞掉了码头的木桩。林岚站在三号栈桥尽头，手里的测光仪屏幕蒙了一层水珠。他用袖口擦了擦，红色数字跳了一下，稳在0.73。比规程低了将近一半。

灯塔在雾里只剩一团昏黄的晕，像泡涨的旧纸。光束转过来时，他数了秒。五秒，比上个月慢了半拍。他合上仪器金属盖，咔嗒声闷在湿气里。

从防水夹层抽出处方本大小的记录簿，纸页边缘已经起毛。他用铅笔在11月23日的栏里写：信号周期偏差+0.5秒，光强衰减至0.73基准单位。笔尖刮擦纸张的沙沙声混进远处货轮的汽笛。他把记录簿塞回去时，左臂袖口滑上去一截，露出手腕内侧一道旧疤，皮肤绷得发白。

回办公楼要穿过鱼市。塑料布棚底下挂着昨夜的残灯，照着地上泛着油光的水渍。腥气钻进鼻腔，混着某种廉价的消毒水味。他在二楼自己那间屋子停住脚——门缝底下塞着东西。

一张对折的码头货运单。展开后是空白的，但纸面有几处凹痕。他举到窗边侧光看，能辨出圆珠笔刻下的字痕：别查灯。纸的右下角有个模糊的指印，沾着点深褐色的污迹，闻起来像铁锈。

他拉开抽屉，取出审计链专用的加密终端。屏幕亮起来时，蓝光照亮下巴的线条。新建条目，编号AL-093。手指在按键上移动：21:47，收到匿名警告纸条，材质为标准货运单，纸背附疑似氧化铁残迹。逐字输入时，终端发出细微的滴答声，像秒针在走。

窗外雾更浓了，路灯变成毛茸茸的光团。他把纸条封进证据袋，标签上写“未经验证信号干扰物证”。袋子边缘沾上了一点灰，是抽屉角落的积尘。

下楼去通讯室时，左肩撞到了门框。他停了一秒，换右手把门推开。老吴正在值班台后头吃盒饭，见他进来，筷子停在半空。

“林调查员。”

“查灯塔信号源登记。过去三个月。”

老吴咽下饭，油腻的指头在键盘上敲。打印机开始吐纸，嗡嗡声里混着老旧电路板的焦糊气。林岚站在旁边等，眼睛盯着窗外。雾里隐约有船影移动，探照灯的光柱被雾切得断断续续。

“所有记录都在这儿。”老吴递过来一叠还热着的纸，“但上周有笔临时维护，申请单没进系统。”

林岚接过去，纸页很轻。他翻到最后一行，签字栏空白。维护项目栏写着：光束校准，特种校准仪调用。批准人是港口管委会的陈副主任。

“这个陈副主任，”林岚把纸叠起来，“上周在港里吗？”

老吴用纸巾擦手，擦得很慢。“出差了。去市里开安全会议，一周。”

林岚把那叠记录夹进自己的记录簿。纸张摩擦的声音很轻，但老吴的勺子碰在饭盒边缘，当的一声。夜班室的空调嗡嗡响着，风口对着林岚的后颈吹，冷气钻进领口。

他走出通讯室时，听见老吴在身后吸溜了一下鼻子。

楼梯间的声控灯坏了。他摸着墙壁往下走，指腹蹭到剥落的墙皮，粗糙的触感。转角处有个窗，雾气从缝隙渗进来，玻璃上凝着水珠。他停在那里，用指尖在水汽上划了一下。指痕很快又被新的湿气填满。

回到办公室，他重新打开终端。在AL-093条目下追加：22:15，确认灯塔维护记录缺失关键审批。校准仪调用未经标准流程。他停顿一下，输入：建议交叉比对陈副主任行程记录与信号异常时间线。

加密完成的提示音是短促的蜂鸣。他关掉终端，屏幕暗下去前，蓝光最后一次掠过墙上的港口平面图。灯塔的位置被红笔圈过，纸边已经卷翘。

林岚把左臂从桌上收回来，袖口拉平整。窗外的雾正在变厚，连最近的码头吊机都看不清了。只有灯塔的光还在转，一圈，又一圈，比规程慢了半拍。
