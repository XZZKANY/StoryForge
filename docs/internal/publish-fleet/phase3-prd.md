## 目标

降低「人肉点后台」成本：**你触发**、可见、可中断。  
在已登录的系统浏览器里打开番茄作者页 + SF 内分步粘贴向导 + 开书后回写向导。

## In Scope（L2）

1. **打开作者后台**：一键用系统默认浏览器打开可配置的番茄作者 URL（不代登）  
2. **分步粘贴向导**：书名 → 简介 → 标签 → 确认已开（每步一键复制，用户自己粘贴）  
3. **开书后回写向导**：向导末步直接「确认已开」写回额度  
4. **今日作战入口**：每本书「开书辅助」  

## Out of Scope

| 不做 | 原因 |
| --- | --- |
| 自动登录 / 存密码 | 红线 |
| DOM 自动填表 / 无人值守提交 | L3，未排期 |
| 打码 / 反检测 / 代理链 | L4，不交付 |
| 爬后台数据 | 用手动校准 |

## 验收

- [ ] 可打开作者后台 URL（真机 shell open / dev 降级提示）  
- [ ] 分步向导可复制 title/blurb/tags  
- [ ] 向导可中断（关闭即停）  
- [ ] 末步可确认已开并占额度  
- [ ] 无登录/打码/自动化浏览器控制代码
