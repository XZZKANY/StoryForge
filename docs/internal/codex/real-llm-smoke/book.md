# Phase 9B 真实 LLM 冒烟 3 章

## 第 1 章 真实冒烟 1

潮气贴在石阶上，鞋底一落，细砂和盐粒就磨出轻响。雾港的夜灯隔着海雾成了一团一团的黄斑，码头边堆着湿麻绳和鱼箱，腥味压在喉口，远处汽笛拖出一线低鸣，像钝刀在铁皮上慢慢刮。

林岚站在防波堤尽头，望着灯塔。

白光每七秒扫一次海面，按港务局旧档，第三轮应有一段短闪提示浅礁。可今晚那一格被拉长了，像有人把节拍硬生生掰歪。她抬腕，表盘冷得发硬，秒针掠过十二的时候，塔顶的光柱正好偏出半寸。

“第三轮长了两秒。”她说。

袖口里的录音钉亮了一下，发出细小的嗒声。她把掌心终端压在栏杆上，指尖划开审计链界面。黑底屏幕映着她的下颌线，海风从领口钻进去，带着锈味。她录入一行简短记录：二十三点十一分，灯塔信号节拍异常，第三轮延长，实测九秒。设备自动抓取时间戳，链上回执在角落跳出一串白码。

她把终端扣回披风内侧，没有解释。

看守塔的老马缩着脖子从值班棚里探出头，手里还端着半杯冷茶。茶面飘着灰，杯口磕掉了一块瓷。

“又来查灯？”老马问，“今晚上潮重，信号飘一点，不稀奇。”

“飘不到两秒。”林岚走近棚门，木门底下渗出煤油和潮木头混在一起的味道，“今晚是谁值中段？”

“照规矩是塔上自动轮转，下面就我一个。”老马把茶杯搁在窗台上，瓷底一碰，发出脆声，“你们审计处上个月不是封过机柜？封条还是新的。”

林岚看了一眼。机柜在棚角，铁门漆皮起泡，铅封的线确实还在，可封泥边缘有一道细裂，裂口里卡着淡灰色粉末。她伸手时左臂顿了一下，动作很短，随即换右手把便携灯压低。光一照，粉末里闪出细碎银屑。

“别碰。”老马往前迈了半步，“港里风大，什么都能吹进去。”

“风吹不进封泥里。”她说。

录音钉又响了一下。她半蹲在机柜前，把裂口、粉末、封线结扣都拍进镜头，再把采样纸贴上去轻轻一按。纸面留下灰银两色的擦痕，带一点焦糊味，像烧过的绝缘皮。她在终端上录第二条：机柜封泥裂，发现疑似导电粉末，现场采样，位置东侧值班棚。提交。回执生成时，屏幕轻轻震了一下。

老马喉结滚了滚，伸手去摸烟盒，摸空了，只好把手塞回袖子里。

“你怀疑有人动塔？”

“有人，或者东西。”林岚起身，目光掠过他袖口。袖边沾着一线黑灰，和采样纸上的颜色近。

海上又转过一轮光。短闪该落下的那一刻，塔顶先是黑了一息，接着蹿出一截发红的火星，像谁在玻璃罩里划了根火柴。火星很快灭掉，只剩一缕细烟，贴着塔身往上爬，在雾里拉成一条浅黑的线。

老马吸了口凉气，茶杯差点脱手。

“灯芯冒烟了？”他声音发飘，“不对，塔灯昨天下午才换——”

“钥匙。”林岚伸手。

“在、在账本下面。”

她把黄铜钥匙串抄进掌心，金属边冰得发麻。值班棚里的挂钟咔嗒咔嗒走着，玻璃罩上全是潮雾。她顺手翻开柜台上的出入账本，纸页潮软，墨迹被手指一蹭就有点散。今夜二十二点四十七分那一行，多了个不该有的签名缩写：R.S. 港务局没有这个轮值员。

她拍照，录入第三条：值班账本异常签注，时间二十二点四十七分，签名缩写R.S.，与排班不符。提交。

“这谁写的？”她问。

老马盯着那行字，嘴唇发白，“我没见过。刚才我去后头烧水，回来账本就在这儿。我以为是白班漏记。”

“你离开多久？”

“十分钟，最多。”

“谁能进棚？”

“港务的人，巡缆工，还有塔上的维修班。”老马咽了口唾沫，“今晚没维修班。”

林岚合上账本，转身往塔基走。石道上潮水拍着底部空洞，咚咚作响，像有人在下面敲门。铁梯扶手满是水珠，手一握，凉意直扎到掌骨。她走得不快，步子稳，每上一层，海雾里的煤烟味就重一点。半腰平台的配电箱门虚掩着，缝里漏出一丝焦黄。

她停下，侧耳听。

箱内有极细的滋滋声，像虫子啃纸。她拿钥匙尖把门顶开半寸，一股烫过的胶皮味立刻扑出来。里面多了一枚拇指大的中继片，铜脚还热，边缘烙着港外黑市常见的三角刻印。原有线路被人并了一路，线皮发黏，指腹一碰就带出黑色。

她把照片、温度读数、刻印细节一并上传。第四条记录落链时，远处传来一阵急促的脚步声，从塔下直冲上来。

“谁在上面？”男人的声音隔着铁梯回荡，“老马，是你吗？”

林岚把中继片夹进证物袋，扣紧封口，袋口齿纹咬合时发出一串细碎响声。她把左臂藏进披风，站到配电箱侧边，借着塔壁的阴影看下去。

来人穿雨布短外套，肩头沾满白色盐霜，脚步却太轻，不像常年在码头搬缆的工人。那人抬头时，灯塔失真的白光正扫过他的脸，额角一道新擦伤亮了一下。

“维修单。”林岚开口，“拿出来。”

那人脚下一顿，扶梯的铁条吱呀一声。海风穿塔而过，把那股烧胶味卷得更浓。下面的老马还在喊：“谁啊？上头什么情况？”

来人没答，手却往衣襟里探。

林岚把终端按亮，审计链回执一条条排在屏幕上，冷白的字映在她指节上。她没往后退，只把证物袋塞进内袋，另一只手压住了腰侧的电击器。

灯塔顶部又冒出一缕烟，贴着雾往海面飘。光柱再一次错拍，白线掠过黑水，把防波堤和那人的鞋尖一起照得发亮。下一秒，铁梯上响起一声急促的金属擦碰。

## 第 2 章 真实冒烟 2

那声金属擦碰一落，林岚先动了。

她侧身贴住塔壁，右手一翻，电击器顶上去，正撞在来人腕骨。蓝白火花炸开一瞬，铁梯上腾起焦糊味。男人闷哼，手里的薄刀当啷掉下两级，磕得栏杆直颤。老马在下头叫了一声，声音被风撕碎。

“别动。”林岚说。

男人咬着牙，另一只手却还往怀里掏。林岚抬膝顶在他腹口，靴底踩住他掉落的刀背。铁梯窄，海风从中空的井道里直灌上来，带着咸湿的凉气和煤烟。那人后肩撞上扶栏，外套里甩出一张折皱的纸，卡在台阶缝里。

“谁派你来的？”

男人喘得发急，额角那道擦伤往下淌水，不知是雾还是汗。“维修单。”他挤出两个字，喉咙里有铁锈味似的粗哑，“上头批的。”

“哪一处上头。”

他眼神一飘，往塔下瞥。林岚顺着那一下看见底层平台外，有一盏手提灯晃了晃，黄光隔着雾，一灭一亮，像在对拍子。

“有人接应。”她说。

男人肩膀猛地一沉，想撞开她往下扑。林岚左臂刚抬半寸就停住，旧伤扯得发紧，她立刻换右肩压上去，把人死死楔在栏杆和塔壁之间。铁锈蹭进掌心，粗粝发疼。她用膝弯一绞，那人脚下一软，半跪在湿台阶上，嘴里吐出一口带腥的气。

老马气喘吁吁爬上半层，脸让风吹得发青，手里攥着一把扳手。

“这、这谁啊？”

“看住下面。”林岚没回头，“有人在塔基。”

老马往下探了一眼，扳手差点滑手。“真有人！提着灯，往西边跑了！”

林岚弯腰，捡起那张纸。纸面被潮气泡软，红章晕开了一圈，抬头写着“港务设备紧急维护许可”，下面的签名不是R.S.，是个更短的记号，只有一个斜钩和一点。纸角沾着黑灰，和机柜封泥里的粉末一个色。她把纸贴到终端下，扫描灯扫过，页面边缘浮出暗纹。

伪章。

她录入第五条：截获伪造维修许可一份，持有人未出示有效身份，文件含异常暗纹与导电残留。提交。屏幕轻震，回执白码一跳一跳，映得她下颌发冷。

“名字。”她问。

男人闭着嘴，牙关咬得咯吱响。

塔顶又是一声轻爆，像玻璃罩里弹开一粒石子。白光这次没有按时扫出，塔身先暗了三拍，紧接着红线从罩内一闪，细烟顺着缝口钻出来。烧胶味更重了，呛得喉口发干。远处海面上有船笛拉长，低沉一线，和失真的灯信号错在一起。

“再不说，灯先烧穿。”老马抹着脸上的水，“林调查员，先上去吧？”

林岚看了眼配电箱里那条并出的线路，又看向男人怀口鼓起的一块。她探手进去，拽出一枚小巧的发报码。金属外壳还温着，侧面也烙着同样的三角刻印。码盘边缘卡着一缕纤维，灰白色，像港务内勤常穿的粗呢袖口。

男人肩背一僵，喉结重重滚了一下。

“谁在收灯塔信号？”她问。

他盯着那枚发报码，鼻翼急促开合，半天才吐出一句：“灰埠仓库。三号冷库后门。”

“谁在那里接线。”

“我没见人。只认灯。”他说，“两长一短，门开。三长，烧塔。”

老马倒吸一口凉气，手里的扳手碰上栏杆，叮地一响。

林岚把发报码和伪造许可一并装进证物袋，封口压实。第六条记录很短：取得便携发报码一枚，关联失真灯信号，指向灰埠仓库三号冷库。提交。

链上回执刚落，她的指尖停了一下。

终端右下角弹出一行新提示：审计节点“港务主档”拒绝同步，签章被上级暂挂。

她盯着那行字，风从披风领口灌进去，背脊立刻发冷。塔顶的滋滋声还在，像有人拿细锯一点点锯铜皮。底下的海水拍空洞，咚，咚，咚，节拍稳得发硬。她又试了一次，屏幕还是那行字，白得刺眼。

“主档挂了？”老马凑近半步，声音压得发虚，“那你的链……”

“本地还在。”林岚说。

她把终端收回去，手却没有立刻松开。指腹压在壳边，压得发白。海风里那股焦味越卷越浓，眼睛先被熏得发涩。她抬手蹭了一下，指节掠过下眼睑，湿意却没有断，顺着脸侧滑下来，落在终端屏上，砸开一小团水印。

老马一下住了口。

林岚背过脸，把左臂更深地藏进披风里，肩线绷得很紧。第二滴跟着落下去，顺着唇角进了嘴里，咸得发苦。她吸了口带烟的冷气，喉头压了一下，开口时声音还是平的：“绳子。”

老马愣了半拍，赶紧把腰间捆缆绳解下来递过去。粗麻绳湿透了，带着鱼腥和海盐，勒在掌心生疼。林岚三下把人反绑在栏杆上，结扣收得死，绳股吃进外套，发出吱嘎的摩擦声。

“看着他。”她说。

“你呢？”

“上塔。断火。”

“可灰埠那边……”

“证物先落链，再去灰埠。”

她抹掉屏上的水痕，重新点开离线见证节点，把六条记录逐条镜像到便携存储里。回执声一连六下，清脆，短促，像钉子一颗颗钉进木板。做完这些，她把存储片塞进靴筒内侧，拉紧披风，转身往上。

铁梯更陡了，扶手上全是冷水，手一握就打滑。她每迈一级，塔顶的热气就更重一点。玻璃罩下那团红光时明时暗，像一口喘不匀的炉子。到顶层门口，门缝里正往外冒烟，带着塑胶熔开的甜腻味。她抬脚踹开门，热浪扑脸，睫毛立刻卷上一层潮热。

灯室中央，旧式旋转机座旁多了个黑盒，黑盒接着一截临时线，线头缠在主控轴上，正随着转动一圈圈磨。每磨一下，灯信号就错一拍。玻璃罩内壁已有细裂，火星在裂口边跳。

林岚扑到主控台前，扳下总闸。啪的一声，白光灭了，整座塔立时只剩海风和烧灼后的细响。黑盒还亮着一点红。她抓起绝缘钳，一口咬住手套边往上扯，右手钳住线头，狠狠一剪。

铜丝弹开，火星溅在铁板上，滋啦一声熄了。

塔外的海面顿时黑下去，只余雾港远处零散的黄灯，浮在潮气里。下方老马的喊声被风送上来：“灭了没有？”

林岚把冒烟的黑盒装进证物袋，扣紧封口，低头看了一眼冷掉的灯座。玻璃罩上那道裂纹斜斜划过，把她的影子截成两段。她抬手按亮终端，录下第七条：塔顶发现信号扰动装置一台，已切断并封存；主灯暂时熄止，待人工复位。提交。

这一次，离线回执先亮了。屏幕角落里，一串白码稳稳停住。她站了两息，转身下塔。

灰埠的方向，雾更厚了。远处有船链拖动的声响，从冷库那边隐约传来，一下一下，像谁在黑里试门。

## 第 3 章 真实冒烟 3

灰埠三号冷库贴着水边，墙皮起泡，门口一盏坏了一半的钨灯在风里轻晃，黄光照着地上的盐霜，像撒开的碎骨。铁链拖过水泥坡道，发出刺耳的刮声，冷库背后的排水沟泛着黑油，腥味里掺一股氨气，冲得鼻腔发辣。

林岚贴着仓墙走，靴底碾过碎冰，咯吱一声。她把终端调成离线见证，屏幕压在掌心，冷白的字映着虎口。三号门外果然挂着一只旧信号灯，玻璃罩里积着雾，灯芯却是新的。

她停在门侧，抬手按了两下灯壳。

两长，一短。

门内“咔”的一响，插销退开半截。还没全开，林岚已经用肩顶进去，右手电击器先送。门后的人刚露出脸，蓝白火花就窜上脖颈。那人后背撞上盐包垛，嘴里吐出一声闷哼，手里的钥匙串飞出去，砸在地上乱响。

仓里冷气开得低，白雾沿地滚。天花板吊着几根黑线，线头全汇进中间一张铁桌。桌上摆着接收机、频谱板，还有一册港务主档的纸质副本，边角卷起，压着一枚铜印。墙角一台小锅炉正在吐热，煤烟混着冷库霜气，味道发涩。

“别出声。”林岚说。

被电倒的人蜷了一下，手还往桌底摸。林岚一脚踢开桌腿下的短铳，金属在水泥地上滑出很远，撞到排水槽边才停。她俯身把人翻过来，灯光照到脸，正是港务局档案室的梁述，袖口粗呢布边磨得起毛，灰白纤维和发报码上卡住的一模一样。

梁述咧着嘴吸气，牙根里有血腥味。“你该在塔上守灯。”

“主档谁挂的。”

梁述偏过头，吐出一口带红的唾沫，落在霜面上，立刻化出一个黑洞。“我只挂链，不碰灯。”

“谁碰灯。”

冷库深处传来一声椅脚摩擦。白雾后面，另一道门慢慢开了。一个中年男人摘下耳罩，手里还拎着热水壶，壶嘴冒汽，哧哧作响。他穿港务总务处的厚呢大衣，胸牌翻着，只露出一个“任”字。

“林调查员。”他把壶放到桌角，壶底烫得铁皮一缩，弹出脆响，“你来得快。”

林岚看了眼那枚铜印，又看他手边的签章盒。“R.S.”

男人抬了下眉梢。“任邵。字母是我留的。账本那行，也是我写的。”

“上级暂挂，也是你。”

“主档在我权限里。”任邵伸手拂掉副本上的霜粒，动作很轻，“灯塔老旧，换一套新导航，港里能省很多钱。可预算卡了半年，总得有人催一催。失真几晚，外海那几家船东就肯签单。你把链落得太快，事情就难看了。”

锅炉里煤块塌了一层，火星从炉门缝里跳出来，烤得空气发干。梁述撑着盐包想坐起，被林岚膝盖一压，又跌回去，呼吸里全是冷白的雾。

“烧塔，也是催？”林岚问。

任邵看着她，没答，伸手把热水倒进茶缸。白汽顶起来，带着陈茶和铁锈味。“灯室那只盒子，原本只错拍，不起火。谁把负载加大了，我也在查。”

“谁能改你留的线。”

“港务里不止我一个人想换灯。”任邵把茶缸往前推了半寸，“主档副本、铜印、接收机，都在这儿。你要的证据，够了。把梁述带走，链我给你放开。塔烧起来，对谁都没好处。”

林岚没碰茶。她抬起终端，对准铁桌、铜印、接收机、纸档，一样一样录进去。屏幕轻震，离线见证回执连着跳出。她又把任邵的脸、胸牌、手边签章盒拍进镜头，声音很平：“第八条。灰埠三号冷库查获港务主档副本、伪签铜印、灯信号接收设备。现场人员任邵、梁述在场。提交。”

任邵的手指在茶缸边停住。“离线链没用。主档不认，明早就是废码。”

“还有见证节点。”林岚说。

她抬腕，点开港外审计公证端口。雾港本地主档灰着，旁边那枚“海事总链”的白标却亮着。她把七条旧记录连同第八条整包推送。进度条一点点往前爬，仓里只剩锅炉喘气和接收机的细鸣。任邵盯着屏幕，嘴角绷直，手慢慢离开了茶缸。

“你接了外链？”

“塔上就接了。”林岚说。

任邵猛地探身去够桌下的总闸。林岚先一步抄起铜印，砸在他腕骨上。骨头闷响一声，热水壶被带翻，滚水泼上频谱板，板上立刻噼啪乱炸，焦臭味直冲出来。任邵捂着手退了半步，后腰撞上锅炉，炉门震开，煤火红了一片。

终端在她掌心里“滴”地一响。

推送完成。

白码稳稳锁住，海事总链的回执长得刺眼，连带时间戳、坐标、见证签章一并落下。林岚把屏幕转给任邵看了一眼，随即收回去，按下存证备份。她把铜印装袋，接收机断电封存，纸档压平拍照，动作一件接一件，没停。

外头传来急促脚步，还有港防队的铜哨。老马的大嗓门混在里面，隔着门板都发飘：“人在里面！我看着梯口那小子，跑不了！”

任邵站着没再动，炉火把他半边脸烤得发红，另一半埋在冷白雾里。“你早知道主档会挂。”

“你留的签名太短。”林岚说，“像怕人认出来。”

门被撞开，冷风卷着海腥味灌进来，锅炉上的热汽一碰就散。港防队冲进仓，靴跟敲在水泥地上，声声发硬。梁述被拎起来时，钥匙串还在地上轻轻打转。任邵抬手遮了一下脸，胸牌终于翻正，名字全露出来。

林岚把最后一只证物袋封口，齿纹咬合，细碎一串响。她抬头看向门外。雾港的海面还黑着，灯塔那边没有再错拍，只有临时检修灯一明一灭，稳稳照着航道。风从冷库口灌进来，带着盐、煤烟和一点烧过后的金属味，贴在脸上发凉。她把终端收回披风内侧，转身往外走。
